# Brain Games
Набор из пяти консольных игр для тренировки мозга. Проект выполнен в рамках обучения на Hexlet

## Требования
Node.js версии 20 или выше

## Установка
```bash
# Клонировать репозиторий
git clone git@github.com:D3lta-dlt/frontend-project-44.git
cd frontend-project-44
# Установить зависимости
make install
```

## Запуск игр
# Приветствие
make brain-games

# Проверка на чётность
make brain-even

# Калькулятор
make brain-calc

# НОД
make brain-gcd

# Арифметическая прогрессия
make brain-progression

# Простое ли число
make brain-prime
